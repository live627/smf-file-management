[color=#228B22][b][size=12pt]What does it do?[/size][/b][/color]
[i]The SMF File Management Tool[/i], aka [i]File Manager/FM[/i] adds a simple yet useful file manager to your administration panel.
It allows you to browse all files and sub folders in your SMF installation folder.
You can edit files, remove files and directories, rename files and directories and chmod files and directories.

[color=#228B22][b][size=12pt]How to use[/size][/b][/color]
To use [i]FM[/i], [url=http://custom.simplemachines.org/mods/index.php?mod=2276]download the package from the SMF mod site[/url], and install it via your [url=http://docs.simplemachines.org/index.php?topic=95]Package Manger[/url].
You will be able to use the file manager without having to do anything else[color=maroon]*[/color]!

[size=1][color=maroon][b]*[/b][/color] = The file manager is found in [i]Admin > Maintenance > File Manager[/i].[/size]
